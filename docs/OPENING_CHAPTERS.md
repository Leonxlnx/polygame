# Opening Chapter Flow

This document defines the first playable minutes so the game opens with a clear rhythm instead of spawning the player into unlocked systems.

## Chapter 1: Morning Cove

The player wakes at the end of a quiet cove path. The first screen only shows health and one objective. No resources, tools, pack, combat, or build UI is visible yet.

1. Walk out of the cove.
2. Follow the ridge path to Edda.
3. Talk to Edda.
4. Gather fallen wood.
5. Return the wood.
6. Craft the wooden pick.
7. Mine loose stone.
8. Build the first shelter.
9. Practice one swing.
10. Gather herbs.
11. Build the first campfire.

## Unlock Rules

- Hearts are always visible.
- Resource UI is hidden until that resource is introduced by the story.
- The hotbar is hidden until the first gathering task.
- The tool slot is hidden until the wooden pick actually exists.
- The pack cannot open while empty.
- Combat stays locked until the campfire stage opens the lower trail.

## Opening Tone

The opening should teach one verb at a time: move, talk, gather, craft, build, swing. Each step introduces only the system the player can use right now.
